# UI Kit / 003 — Mode-Preserving Spec

## Product role
003 is the **marketing / entrance expression layer** for the UI Kit concept.

It is not:
- docs home
- component browser
- token inspector
- dashboard
- changelog screen

It is:
- a campaign poster for the kit
- the first authored encounter with the system
- a launch mood surface before the utility layers open

## Core rule
**Preserve the original Julie Doucet single-poster demo mode.**

The page must still read as:
- one tall printed object in a dark field
- oversized staggered condensed title islands
- sparse note fragments
- pink collage forms + red hand marks
- restrained pointer-inspection behavior

If it starts feeling like a normal landing page, it fails.

## UI Kit translation
What 003 needs to communicate:
1. the name `UI Kit`
2. that this is a shippable system of parts/tokens/modules
3. that the viewer is about to enter a larger library

Do this with **very little text**.

Allowed phrases:
- UI KIT
- ENTRY POSTER
- TOKENS / PARTS
- MODULE DRIFT
- SYSTEM LAUNCH
- SHIP READY
- type / motion / layout / release

Forbidden:
- feature blocks
- onboarding copy
- product paragraphs
- CTA stacks
- docs-style descriptions

## Composition constraints
The layout should visibly preserve the source poster logic:
- tall poster ratio
- large title clusters distributed down the sheet
- left/right alternation
- small note copy living in the gaps
- collage shapes acting as structural anchors
- scribbles behaving like connective tissue

Do not flatten into:
- a centered hero
- a balanced editorial poster
- a grid of marketing modules

## Interaction
Pointer interaction is required, but physical and restrained:
- poster tilts
- inspection light follows pointer
- internal layers drift slightly

No elastic or app-like motion.

## Acceptance
Pass only if:
1. it reads immediately as a **single poster-object**
2. it sells **entry mood**, not product explanation
3. text remains extremely sparse
4. the original 003 placement logic is still obvious
5. it feels like a believable front-door expression layer for UI Kit
