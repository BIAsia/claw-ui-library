# claw-ui-library

`claw-ui-library` is a reconstruction-driven UI library and probe space.

## Agent style-spec rule for this repo

For `003 / 004 / 005`-style spec work, the goal is **mode-preserving concept adaptation**.

That means:
- keep the original demo's recognizable interaction/composition grammar
- adapt the concept **inside that mode**
- do **not** let the result drift into a generic website just because the concept changes

Examples:
- **003** should still feel like an interactive poster/card surface with sparse text and pointer-reactive motion
- **004** should still behave like a browse-first catalog/grid system
- **005** should still read like a magazine/editorial front page rather than a standard product landing page

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
